In the past few weeks of my journey with Stutern, I have been taught in-depth on HTML and CSS and what they entail. HTML(Hyper Text Markup Language) is the standard markup language for documents designed to be displayed in a web browser. This is basically the foundation and structure of a webpage. CSS(Cascading Style Sheets) is used to describe styles(as the name implies) for presenting documents written in a markup language i.e HTML, or XML. I learnt how box model works, with the parameters governing a box model. I was also taught CSS inheritance, Specificity Hierarchy, CSS selectors, types and properties.

Padding: This is the space between the element's border and the related content inside it. It determines how elements look and sit within a container. Simply put, padding property controls the space inside an element.

Margin: This is the space around an element's border. Margin property controls the space outside an element.

Border: The CSS border property allows you to specify the style, width, and color of an element's border.
